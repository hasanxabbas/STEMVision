# STEMVision

## AI-Powered Accessible Learning Platform for STEM Students

STEMVision is a MERN Stack application that combines Artificial Intelligence and Accessibility to improve STEM education.

## Features

### Teacher
- Upload Notes (PDF, PPT, Images)
- Manage Subjects
- Upload Assignments
- Send Announcements

### Student
- AI Tutor
- Subject-wise Learning
- Voice Learning
- AI Quiz Generator
- Learning History

## Accessibility

- TalkBack Compatible
- Voice Navigation
- High Contrast Mode
- Large Touch Targets
- Text-to-Speech
- Speech-to-Text

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- Gemini API
- OCR
- RAG
- Text-to-Speech
